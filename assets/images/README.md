## Image Guidelines:

### Profile Images:
- Use high-quality, professional photos
- Ensure good lighting and clear focus
- Square format works best for profile pictures
- Consider using a consistent background or style

### Project Screenshots:
- Capture clean, representative screenshots of your projects
- Use consistent dimensions for uniformity
- Show the most impressive or representative views
- Consider adding mockups for mobile responsiveness

## Optimization Tips:
- Compress images to reduce file size (aim for <200KB per image)
- Use WebP format when possible for better performance
- Maintain aspect ratios to prevent distortion
- Consider using services like TinyPNG for compression
