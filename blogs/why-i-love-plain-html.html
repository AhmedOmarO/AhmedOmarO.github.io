Why I Love Plain HTML

HTML is still my favourite way to share an idea quickly. I can sketch a layout, publish a note, or test a concept without waiting for a build step.

Less tooling, more writing

When I write in HTML the browser shows me exactly what I created. That immediacy keeps me focused on the story instead of the stack.

Great with Markdown

Some posts start as Markdown in my notes app. When I'm ready, I polish the structure with HTML and ship it as-is. Now this site supports both so I can choose what fits the moment.
